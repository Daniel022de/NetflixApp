# Aplicativo protótipo Netflix

Este aplicativo foi desenvolvido com a finalidade de estudar alguns recursos importantes no desenvolvimento mobile.

## Recursos Principais Aprendidos  

- **Life cycle Activity**
- **API Restful**
- **Threads**
- **Arquitetura MVC**
- **RecyclerView**
